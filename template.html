關於「每週自動更新」

這個頁面本身是一份靜態網頁，無法自己連線到網路抓新聞或自己排程執行——它只會顯示寫入頁面當下的內容。要做到「真正每週自動更新」，需要一個會定期執行的程式去抓資料、寫回頁面。以下是三種常見做法，從最簡單到最完整：

方法一

每週請我重新生成

每週回來這個對話，請我搜尋當週新聞並更新這份頁面，幾分鐘就能拿到新版本。不需要寫程式，但要手動觸發。

方法二

排程腳本 + 新聞 API

寫一個 Python／Node 腳本，串接新聞 API（如 NewsAPI、GNews）或 RSS 來源，搭配 cron job 或 GitHub Actions 每週執行一次，把結果寫入此頁面的 JSON 資料再部署。

方法三

串接 Claude API 自動摘要

排程腳本呼叫 Claude API（搭配 web search 工具）每週自動搜尋三地新聞並生成摘要文字，再寫入網頁的資料來源，全程免人工。
